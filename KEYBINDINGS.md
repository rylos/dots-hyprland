# Hyprland Keybindings

## Shell & Interface

| Keybind | Action |
|---------|--------|
| `Super` | Toggle overview/launcher |
| `Super + Tab` | Toggle overview (alternative) |
| `Super + A` | Toggle left sidebar |
| `Super + N` | Toggle right sidebar |
| `Super + /` | Toggle cheatsheet |
| `Super + K` | Toggle on-screen keyboard |
| `Super + M` | Toggle media controls |
| `Super + J` | Toggle bar |
| `Ctrl + Alt + Delete` | Toggle session menu |
| `Ctrl + Super + R` | Restart widgets |

## Utilities

| Keybind | Action |
|---------|--------|
| `Super + V` | Clipboard history |
| `Super + .` | Emoji picker |
| `Super + Shift + S` | Screen snip |
| `Super + Shift + T` | OCR (character recognition) |
| `Super + Shift + C` | Color picker |
| `Print` | Screenshot to clipboard |
| `Ctrl + Print` | Screenshot to clipboard & save |
| `Super + Alt + R` | Record region (no sound) |
| `Super + Shift + Alt + R` | Record screen (with sound) |
| `Ctrl + Super + T` | Wallpaper selector |

## Window Management

### Focus & Movement
| Keybind | Action |
|---------|--------|
| `Super + ←/↑/→/↓` | Focus window in direction |
| `Super + Shift + ←/↑/→/↓` | Move window in direction |
| `Super + Q` | Close window |
| `Super + Shift + Alt + Q` | Force kill window |
| `Alt + F4` | Close window (Windows style) |

### Window States
| Keybind | Action |
|---------|--------|
| `Super + Alt + Space` | Toggle float/tile |
| `Super + D` | Maximize |
| `Super + F` | Fullscreen |
| `Super + Alt + F` | Fullscreen spoof |
| `Super + P` | Pin window |

### Window Sizing
| Keybind | Action |
|---------|--------|
| `Super + ;` | Decrease split ratio |
| `Super + '` | Increase split ratio |
| `Super + Mouse Left` | Move window |
| `Super + Mouse Right` | Resize window |

## Workspace Management

### Switching Workspaces
| Keybind | Action |
|---------|--------|
| `Super + 1-9,0` | Focus workspace 1-10 |
| `Super + Page Up/Down` | Focus previous/next workspace |
| `Super + Mouse Scroll` | Focus previous/next workspace |
| `Ctrl + Super + ←/→` | Focus left/right workspace |
| `Super + S` | Toggle scratchpad |

### Moving Windows to Workspaces
| Keybind | Action |
|---------|--------|
| `Super + Alt + 1-9,0` | Send to workspace 1-10 |
| `Super + Alt + S` | Send to scratchpad |
| `Super + Shift + Mouse Scroll` | Send to previous/next workspace |
| `Super + Alt + Page Up/Down` | Send to previous/next workspace |
